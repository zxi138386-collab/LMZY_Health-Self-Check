1

内容生产

25分
你的账号有清晰的选题SOP吗？——选题有计划、有排期，不是老板想起来才拍一条。
0 = 完全随机发 / 5 = 有月度选题日历和SOP
每条内容的"人设印记"清晰吗？——观众看完知道你是谁、你帮谁、你能解决什么问题。
0 = 没有辨识度 / 5 = 人设鲜明，观众一眼认出
有一条"置顶内容"能准确告诉新访客：你是谁、你帮谁、怎么找你？
0 = 没有置顶 / 5 = 有清晰的欢迎页/置顶视频
内容发布频率稳定吗？——每周至少保持固定频次，不"三天打鱼两天晒网"。
0 = 不规律 / 5 = 每周3条以上且节奏稳定
每条内容末尾有明确的"行动引导"吗？——告诉观众下一步该做什么（关注/私信/评论）。
0 = 没有CTA / 5 = 每一条都有清晰且有效的CTA
2

私域承接

25分
加微信后5分钟内，有没有自动发送"欢迎信息+价值主张"？
0 = 加了不说话 / 5 = 有完善的自动欢迎流程
72小时内是否有节奏地推进关系——从认识、到互动、到首次价值交付？
0 = 加了就不管了 / 5 = 有完整的72小时承接SOP
朋友圈有"培育剧本"吗？——不是随便转发，而是有节奏地展示专业度、案例、客户证言。
0 = 朋友圈随便发 / 5 = 有朋友圈内容日历和剧本
客户问完价没下文之后，有跟进流程吗？
0 = 不回就放弃 / 5 = 有标准化的二次触达流程
老客户有转介绍机制吗？——满意的客户知道推荐你能得到什么。
0 = 没有转介绍 / 5 = 有成熟的转介绍激励机制
3

投流获客

25分
投流之前，有没有先用自然流量验证过"哪类内容能带来询盘"？
0 = 没验证直接投 / 5 = 有数据验证的投流策略
每条投放素材都对应一个明确的转化路径吗？——从看到广告到加上微信，路径是通的。
0 = 投了不知道去哪 / 5 = 每个素材都有清晰的转化链路
你能说出每条线索的获客成本吗？——不是大概，是能算到个位数的数字。
0 = 不知道 / 5 = 有精确的获客成本追踪
投流素材多久迭代一次？——有A/B测试机制吗？
0 = 一条素材跑到底 / 5 = 每周迭代，有系统化的A/B测试
搜索排名：潜在客户搜你的行业关键词，你能排在第几位？
0 = 搜不到 / 5 = 前三位
4

组织能力

25分
团队有专人负责新媒体吗？——不是"行政兼着发一下"。
0 = 没人负责 / 5 = 有专职团队且岗位清晰
操盘手/负责人有能力独立完成"选题→拍摄→发布→复盘"闭环吗？
0 = 完全依赖老板 / 5 = 能力完整，可独立运转
有内容素材库吗？——过往内容是否归档、可检索、可复用？
0 = 散落在手机相册 / 5 = 有结构化的素材库
每周有固定复盘会议吗？——看数据、找问题、定下周计划。
0 = 不做复盘 / 5 = 每周固定复盘+数据追踪
老板本人愿意出镜或参与内容吗？——新媒体不是"交给下面人"就能做好的。
0 = 老板不管 / 5 = 老板亲自参与且持续输出
